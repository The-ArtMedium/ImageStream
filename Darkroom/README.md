## What is the Darkroom?

The Darkroom is the education folder of the ImageStream open suite.

Two free crash courses — SQL and Python — built for anyone preparing to work with data. Not a bootcamp. A warm-up for one. Or a starting point for anyone who wants to understand data before the intensity begins.

The photography theme is just the vehicle. The destination is confidence with data — regardless of your field, age, or background.

**Built for human eyes.** Warm dark background. Large text. High contrast. No grey on slightly-lighter-grey. No squinting at pale code on a white screen.

## The Dataset

All courses use the same original dataset — created for this project, free to use and share.

| File | Contents |
|---|---|
| `photographers.csv` | 20 photographers from 20 countries — name, country, specialty, rating, camera |
| `shoots.csv` | 103 shoots — location, year, lens, fee, client rating |
| `equipment.csv` | 68 equipment records — type, brand, model, purchase year, price |

## How to Use

1. Download all files into one folder on your computer
2. Open `Before_You_Start.html` in any browser
3. Read it fully — it changes how the lessons land
4. Open `SQL_Crash_Course_Dark.html` or `Python_Crash_Course_Dark.html`
5. The CSV files must be in the same folder for the Python exercises to work

No installation. No account. No internet required after download.
