Variables & Data Types

The building blocks of every Python program

A variable is a name that holds a value. You create it by writing the name, an equals sign, and the value. Python figures out the type automatically.

No need to declare types

In some languages you must say "this is a number" before using it. In Python you just write the value and Python understands. This makes it faster to write.

Basic variables — photography examples

Python
# String — text in quotes photographer_name = "Amara Diallo" country = "Senegal" specialty = "Wildlife" # Integer — whole number years_experience = 22 total_shoots = 779 # Float — number with decimal rating = 3.7 shoot_fee = 750.00 # Boolean — True or False only is_available = True has_drone_licence = False print(photographer_name) print("Rating:", rating) print("Years active:", years_experience)
Output
Amara Diallo Rating: 3.7 Years active: 22

The four main data types

TypeExampleUse for
str"Amara Diallo"Names, text, labels
int22Whole numbers — years, counts
float3.7Decimals — ratings, fees, prices
boolTrue / FalseYes/No decisions

Checking the type and doing arithmetic

Python
print(type(photographer_name)) # <class 'str'> print(type(rating)) # <class 'float'> # Arithmetic with variables photos_taken = 744 photos_exported = 128 kept_percentage = (photos_exported / photos_taken) * 100 print(round(kept_percentage, 1), "%")
Output
<class 'str'> <class 'float'> 17.2 %
Naming Convention

Python variable names use lowercase with underscores — called snake_case. Write photographer_name not PhotographerName. This is the Python standard and everyone will expect it.

Lists, Dicts & Tuples

Storing collections of data in one place

A single variable holds one value. But what if you need to store 20 photographer names? Or a photographer's full profile? Python has three main collection types for this.

List — an ordered collection

A list holds multiple values in order. Use square brackets.

Python
specialties = ["Wildlife", "Street", "Portrait", "Documentary", "Landscape"] # Access by position — counting starts at 0 print(specialties[0]) # Wildlife print(specialties[2]) # Portrait print(specialties[-1]) # Landscape — last item # Useful list operations print(len(specialties)) # 5 specialties.append("Architecture") # add to end print(len(specialties)) # 6

Dictionary — key and value pairs

A dictionary stores data with labels. Like a real dictionary — you look up a word (key) and get the definition (value). Use curly braces.

Python
photographer = { "name": "Priya Nair", "country": "India", "specialty": "Portrait", "years_experience": 23, "rating": 4.6, "available": True } # Access values by key print(photographer["name"]) # Priya Nair print(photographer["rating"]) # 4.6 # Update a value photographer["rating"] = 4.8 # Add a new key photographer["camera"] = "Nikon Z7 II"

Tuple — fixed, unchangeable list

A tuple is like a list but cannot be changed after creation. Use round brackets. Good for data that should never change.

Python
# GPS coordinates — should never change lagos_coords = (6.5244, 3.3792) nairobi_coords = (-1.2921, 36.8219) print(lagos_coords[0]) # 6.5244 — latitude print(lagos_coords[1]) # 3.3792 — longitude
TypeSyntaxChangeable?Best for
list[a, b, c]YesCollections that grow or change
dict{"key": value}YesNamed fields — like a profile
tuple(a, b, c)NoFixed data — coordinates, constants

Loops & Conditionals

Making decisions and repeating actions

Two of the most important tools in any programming language — if/else for making decisions, and for loops for repeating actions.

if / elif / else — making decisions

Python
rating = 4.6 if rating >= 4.5: print("Top rated photographer") elif rating >= 4.0: print("Well rated photographer") elif rating >= 3.0: print("Average photographer") else: print("Needs improvement")
Output
Top rated photographer
Indentation is everything in Python

The code inside an if block must be indented — 4 spaces or one Tab. Python uses this indentation to know what belongs inside the if. If you forget the indent, Python will give an error.

for loop — repeating over a list

Python
photographers = [ {"name": "Amara Diallo", "rating": 3.7}, {"name": "Priya Nair", "rating": 4.6}, {"name": "Sarah Oduya", "rating": 4.8}, {"name": "Carlos Mendez", "rating": 3.8}, ] for p in photographers: if p["rating"] >= 4.5: print(p["name"], "— Top Rated ⭐") else: print(p["name"])
Output
Amara Diallo Priya Nair — Top Rated ⭐ Sarah Oduya — Top Rated ⭐ Carlos Mendez

for loop with range — counting

Python
# Print shoot numbers 1 to 5 for i in range(1, 6): print("Shoot number", i)

Functions

Write code once, use it everywhere

A function is a reusable block of code with a name. Instead of writing the same logic 10 times, you write it once as a function and call it whenever you need it.

Syntax
def function_name(parameter): code goes here return result

A simple function

Python
def kept_percentage(taken, exported): """Calculate what percentage of photos were kept.""" return round((exported / taken) * 100, 1) # Call the function with different shoots print(kept_percentage(744, 128)) # 17.2 print(kept_percentage(1200, 300)) # 25.0 print(kept_percentage(200, 45)) # 22.5
Output
17.2 25.0 22.5

Function with a conditional

Python
def rate_photographer(rating): """Return a label based on photographer rating.""" if rating >= 4.5: return "⭐ Top Rated" elif rating >= 4.0: return "✓ Well Rated" elif rating >= 3.0: return "— Average" else: return "↓ Below Average" ratings = [4.6, 3.7, 4.1, 4.8, 3.5] for r in ratings: print(r, "—", rate_photographer(r))
Output
4.6 — ⭐ Top Rated 3.7 — — Average 4.1 — ✓ Well Rated 4.8 — ⭐ Top Rated 3.5 — — Average
Docstrings

The text in triple quotes right after def is called a docstring. It explains what the function does. Always write one — your future self will thank you when you read your code 6 months later.

Working with Data

Reading CSV Files

Load real data from photographers.csv

A CSV file (Comma Separated Values) is a simple text file where each row is a record and commas separate the columns. It is the most common format for sharing data. Python's built-in csv module reads them easily.

Read photographers.csv

Python
import csv with open("photographers.csv", "r") as file: reader = csv.DictReader(file) for row in reader: print(row["name"], "—", row["country"], "—", row["specialty"])
Output (first 4 rows)
Amara Diallo — Senegal — Wildlife Carlos Mendez — Brazil — Street Priya Nair — India — Portrait James Okonkwo — Nigeria — Documentary
DictReader vs reader

csv.reader gives you rows as plain lists — row[0], row[1]. csv.DictReader gives you rows as dictionaries — row["name"], row["country"]. DictReader is almost always better because names are clearer than numbers.

Load all rows into a list

Python
import csv photographers = [] with open("photographers.csv", "r") as file: reader = csv.DictReader(file) for row in reader: photographers.append(row) print("Total photographers:", len(photographers)) print("First entry:", photographers[0]["name"]) # Find all Wildlife photographers wildlife = [p for p in photographers if p["specialty"] == "Wildlife"] print("Wildlife photographers:", len(wildlife))
Output
Total photographers: 20 First entry: Amara Diallo Wildlife photographers: 3

Pandas & DataFrames

The most powerful data tool in Python

Pandas is a Python library built for data analysis. It loads your CSV into a DataFrame — a table you can query, filter, sort, and calculate on with just one line. It replaces hundreds of lines of manual CSV code.

Load the data

Python
import pandas as pd # Load each CSV file into a DataFrame photographers = pd.read_csv("photographers.csv") shoots = pd.read_csv("shoots.csv") equipment = pd.read_csv("equipment.csv") # Quick look at the data print(photographers.shape) # (20, 8) — 20 rows, 8 columns print(photographers.head()) # first 5 rows print(photographers.columns) # column names

describe() — instant statistics

Python
print(photographers[["rating", "years_experience", "total_shoots"]].describe())
Output
rating years_experience total_shoots count 20.00 20.00 20.00 mean 4.12 13.55 369.10 std 0.43 7.12 259.23 min 3.50 2.00 20.00 25% 3.80 7.50 133.50 50% 4.15 13.50 337.00 75% 4.50 21.25 578.50 max 4.80 25.00 779.00

Access columns and rows

Python
# One column — returns a Series print(photographers["name"]) # Multiple columns — returns a DataFrame print(photographers[["name", "country", "rating"]]) # Basic calculations on a column print("Average rating:", photographers["rating"].mean()) print("Highest rating:", photographers["rating"].max()) print("Total shoots:", photographers["total_shoots"].sum())
import pandas as pd

The "as pd" part gives pandas a short nickname. Instead of typing pandas.read_csv() every time, you type pd.read_csv(). This is a universal Python convention — everyone uses pd for pandas.

Filtering & Grouping

Ask real questions of real data

This is where pandas becomes truly powerful. Filtering and grouping lets you answer real questions in one or two lines — questions that would take 20 lines of CSV code.

Filtering rows

Python
import pandas as pd photographers = pd.read_csv("photographers.csv") # Photographers with rating above 4.5 top_rated = photographers[photographers["rating"] > 4.5] print(top_rated[["name", "country", "rating"]]) # Portrait photographers only portrait = photographers[photographers["specialty"] == "Portrait"] # Multiple conditions — use & for AND, | for OR experienced_wildlife = photographers[ (photographers["specialty"] == "Wildlife") & (photographers["years_experience"] > 10) ]

groupby — summary statistics per group

Python
# Average rating per specialty by_specialty = photographers.groupby("specialty")["rating"].mean() print(by_specialty.sort_values(ascending=False))
Output
specialty Documentary 4.45 Portrait 4.40 Landscape 4.35 Wildlife 4.20 Street 3.90 Name: rating, dtype: float64

sort_values and value_counts

Python
# Sort by rating, highest first ranked = photographers.sort_values("rating", ascending=False) print(ranked[["name", "rating"]].head(5)) # Count photographers per country print(photographers["country"].value_counts())

Visualisation

Matplotlib Charts

Turn numbers into visual stories

Matplotlib is Python's standard charting library. Combined with pandas, you can go from raw CSV to a meaningful chart in under 10 lines of code.

Bar chart — photographers per specialty

Python
import pandas as pd import matplotlib.pyplot as plt photographers = pd.read_csv("photographers.csv") # Count per specialty counts = photographers["specialty"].value_counts() # Create bar chart plt.figure(figsize=(10, 5)) plt.bar(counts.index, counts.values, color="#6bcf8a") plt.title("Photographers by Specialty") plt.xlabel("Specialty") plt.ylabel("Count") plt.tight_layout() plt.savefig("specialty_chart.png") plt.show()

Histogram — distribution of ratings

Python
plt.figure(figsize=(8, 5)) plt.hist(photographers["rating"], bins=10, color="#e8b84b", edgecolor="#1a1a14") plt.title("Distribution of Photographer Ratings") plt.xlabel("Rating") plt.ylabel("Number of Photographers") plt.tight_layout() plt.savefig("ratings_histogram.png") plt.show()

Line chart — shoots per year

Python
shoots = pd.read_csv("shoots.csv") by_year = shoots.groupby("year").size() plt.figure(figsize=(10, 5)) plt.plot(by_year.index, by_year.values, marker="o", color="#7eb8f7", linewidth=2) plt.title("Number of Shoots Per Year") plt.xlabel("Year") plt.ylabel("Shoots") plt.grid(True, alpha=0.2) plt.tight_layout() plt.savefig("shoots_per_year.png") plt.show()
