
# 🎬 LocalClip

**Simple. Local. Yours.**

One job. Done right.

## 🚀 Quick Start

### Installation

1. **Download LocalClip**
   - Get the ZIP from GitHub
   - Unzip anywhere on your computer

2. **Install Python 3.8+**
   - Download from [python.org](https://www.python.org/downloads/)
   - Check "Add Python to PATH" during install

3. **Install FFmpeg**
   - **Windows:** [ffmpeg.org](https://ffmpeg.org/download.html)
   - **Mac:** `brew install ffmpeg`
   - **Linux:** `sudo apt install ffmpeg`

4. **Run the installer**
   - **Windows:** Double-click `install.bat`
   - **Mac/Linux:** Run `./install.sh` in Terminal

5. **Launch LocalClip**
   - **Windows:** Double-click `run.bat`
   - **Mac/Linux:** Run `./run.sh`

## 📖 How to Use

### Basic Trimming

1. **Import Video**
   - Click "Import Video" or drag and drop
   - Supports: MP4, AVI, MOV, MKV, WebM

2. **Set Start Point**
   - Scrub timeline to desired start
   - Click "Mark Start" or press `I`

3. **Set End Point**
   - Scrub to desired end
   - Click "Mark End" or press `O`

4. **Export Clip**
   - Click "Export Clip"
   - Choose location and filename
   - Done!

### Tips

- **Keyboard shortcuts:**
  - `Space` - Play/Pause
  - `I` - Mark In point
  - `O` - Mark Out point
  - `Arrow keys` - Frame-by-frame navigation

- **Multiple clips:**
  - Export one clip
  - Reset markers
  - Mark new section
  - Export again
