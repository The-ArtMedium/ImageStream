# 🎬 LocalEdit

**Simple. Local. Yours.**

A lightweight, privacy-focused video editor for creators who want to keep control of their content. No cloud uploads, no watermarks, no subscriptions—just clean, simple editing on your own machine.

---

## 🎯 Why LocalEdit?

**Own Your Content**
- No watermarks branding your work
- Your files never leave your computer
- No terms of service to worry about
- Export and share on your terms

**Keep It Simple**
- Just 4 layers: Video, Images, Text, Audio
- Drag and drop interface
- Perfect for quick promos, music videos, and social content
- No bloat, no overwhelming menus

**Stay Private**
- Zero cloud storage
- No account required
- No analytics tracking your work
- Your creative process is yours alone

**Go Global**
- Multilingual interface
- Support for all major scripts (Latin, Arabic, CJK, Devanagari, etc.)
- Right-to-left text support
- Unicode and emoji ready

## ✨ Features

### 4-Layer Timeline
- **Layer 1 (Video/Image)**: Background foundation
- **Layer 2 (Image Overlays)**: Graphics, logos, posters with blend modes
- **Layer 3 (Text)**: Titles, captions, subtitles with custom styling
- **Layer 4 (Audio)**: Your music, voiceover, or sound effects

### Core Capabilities
- Drag & drop media files
- Trim and position clips
- Text editor with font controls
- Image blending modes (normal, multiply, overlay, screen)
- Audio volume and fade controls
- Export to MP4
- Cross-platform (Windows, Mac, Linux)

### Coming Soon
- Simple transitions (crossfade, cut)
- Text animations
- Audio fade in/out

## 🚀 Quick Start

### Installation

**Requirements:**
- Python 3.8 or higher
- FFmpeg (for video processing)

**Install:**
```bash
# Clone the repository
git clone https://github.com/yourusername/LocalEdit.git
cd LocalEdit

# Install dependencies
pip install -r Requirements.txt

# Run LocalEdit
python Src/main.py
```

### Your First Edit

1. **Launch LocalEdit**
   ```bash
   python Src/main.py
   ```

2. **Add Your Media**
   - Drag a video or image to Layer 1 (background)
   - Add overlay images to Layer 2
   - Create text captions in Layer 3
   - Drop your audio file into Layer 4

3. **Arrange & Edit**
   - Position elements on the timeline
   - Adjust text size, color, and position
   - Set blend modes for images
   - Trim audio as needed

4. **Export**
   - Click "Export to MP4"
   - Choose your destination
   - Done! Your video is ready to share
