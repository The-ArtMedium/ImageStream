# Changelog

All notable changes to LocalEdit will be documented here.

## [0.1.0] - 2026-02-17

### Added
- 4-layer timeline (Video, Image, Text, Audio)
- Import video, image, audio files
- Export to MP4
- Multilingual support (English, Spanish, Portuguese, Hindi, Arabic)
- Language switcher in menu
- Save/Load projects (.lep format)
- Dark theme UI
- Windows installer (install.bat)
- Mac/Linux installer (install.sh)
- Windows launcher (run.bat)
- Mac/Linux launcher (run.sh)

### Coming Soon

- Audio fade in/out

- More languages
