# How to Install LocalEdit

## Step 1: Download
Download the LocalEdit ZIP file and unzip it to any folder on your computer.

## Step 2: Install Python
If you don't have Python installed:
- Go to https://www.python.org/downloads/
- Download Python 3.8 or higher
- **Windows:** Check "Add Python to PATH" during installation!
- **Mac/Linux:** Python may already be installed

## Step 3: Install FFmpeg
- **Windows:** Download from https://ffmpeg.org/download.html
- **Mac:** Open Terminal and type: brew install ffmpeg
- **Linux:** Open Terminal and type: sudo apt install ffmpeg

## Step 4: Install LocalEdit
- **Windows:** Double-click install.bat
- **Mac/Linux:** Open Terminal in LocalEdit folder and type: ./install.sh

## Step 5: Run LocalEdit
- **Windows:** Double-click run.bat
- **Mac/Linux:** Open Terminal and type: ./run.sh
