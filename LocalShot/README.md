# 📸 LocalShot

**Image Editor for Everyone**
Part of the ImageStream Local Suite by The Art Medium

Simple. Offline. Private. Yours.

---

## Features

- 📁 Open folder — browse all images as thumbnails
- 🖼 Click any thumbnail to open it
- ☀️ Simple / Advanced mode toggle
- **Basic:** Brightness, Contrast, Exposure
- **Advanced Tone:** Whites, Blacks
- **Color:** White Balance (Temperature + Tint)
- **Detail:** Sharpening, Dehaze
- ✂️ Crop by dragging
- ⊞ Resize
- ↺↻ Rotate & Flip
- 💾 Export — JPG, PNG, WebP, BMP, TIFF
- 🌍 7 languages — English, Español, Français, Português, العربية, 中文, हिन्दी

---

## Install

**Windows:**
```
install.bat
run.bat
```

**Mac / Linux:**
```
chmod +x install.sh run.sh
./install.sh
./run.sh
```

**Requirements:** Python 3.8+ · Pillow · NumPy
