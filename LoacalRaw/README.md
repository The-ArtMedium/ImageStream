# LocalRAW v1.0.0

LocalRAW is a fully offline, privacy-focused RAW image editor designed for photographers who need speed, simplicity, and independence.

No cloud.
No subscriptions.
No uploads.
Your images stay on your machine.

---

## 🌍 Purpose

LocalRAW was created to support photographers working in:

- Low-bandwidth regions
- Rural areas
- Fast broadcast environments
- Independent journalism
- Sports photography workflows

The goal is to provide a lightweight, expandable, local editing tool.

---

## ✨ v1.0 Features

- Folder import
- Thumbnail grid view
- RAW support (CR2, NEF, ARW, DNG)
- Dark theme
- Modular architecture
- Export to JPG and PNG
- Multi-language structure (5 languages ready)
- Fully local operation

---

## 🖥 System Requirements

- Windows 10+ or macOS
- Python 3.10+ (only if running from source)
- 8GB RAM recommended
- Modern CPU

Standalone builds will not require Python.
